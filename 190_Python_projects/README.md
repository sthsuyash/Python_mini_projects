## Run any project

1. Clone the repository

   ```git
   git clone
   ```

2. Change into the directory

   ```bash
   cd 190_Python_projects
   ```

3. Run the program

   ```bash
   python3 <file_name>.py
   ```

   or

   ```pwsh
   python <file_name>.py
   ```

   OR

   copy the code and run it in your IDE
